# Cat & Kid Goat Storybook

- 75 pages, each with short story text
- 15pt font size
- 0.125 inch safe zone (9pt) on all sides
- Attractive UI and downloadable PDF via `@react-pdf/renderer`

## Local

```bash
npm install
npm run build
npm start
```

Open http://localhost:3000 and click Download 75-page PDF.

## Deploy

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-3d44ec0a
```
